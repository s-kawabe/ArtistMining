# アーティスト発掘アプリ「ArtistMining」

## コンセプト
自分の好きなアーティストを世に広めたいユーザ
×
自分に合うアーティストを探したいユーザ

のマッチングアプリ。
世の中をもっと音楽で楽しく。

### 技術
- Ruby v2.5.3
- Rails v5.2.4.3
- Javascript
- Bootstrap
- gem CarrierWave

### 機能
- ユーザ登録
- ログイン
- ユーザ個人ページ表示
- ユーザサムネイル登録
- ユーザ情報変更
- アーティスト閲覧、投稿
- アーティストサムネイル登録
- アーティスト検索
- コメント機能
- 非同期いいね
